以下の URL のドキュメントを読み込み、内容の理解確認クイズを作成してください。

URL: $ARGUMENTS

## 手順

1. WebFetch ツールで URL のコンテンツを取得する
2. 内容を理解したうえで、重要なポイントを5問の4択クイズにする
3. 全問を一度に提示し、ユーザーに回答を求める
4. ユーザーが回答したら、各問の正誤・解説を返す

## クイズのルール

- 4択形式（A/B/C/D）
- 重要な概念・使い分け・注意点を問う
- 全問出題してからまとめて採点する
- 正誤には必ずドキュメントの根拠を添えて解説する
